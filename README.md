## ‚¨° Overview

**Madhu.dev** is the personal developer portfolio of **Madhu Shankar Kumar**, a B.Tech CSE student at GLA University specializing in Data Engineering, Java Development, and AI & Data Science.

This portfolio was crafted entirely from scratch ‚ÄĒ no Bootstrap, no Tailwind, no UI kits. Every animation, every layout, every interaction is hand-engineered using Vanilla CSS3 and TypeScript. The visual identity is built around a **Minimalist Matte Slate** theme: pitch-black backgrounds, matte graphite surfaces, metallic silver gradients, and glassmorphism card designs with floating ambient orbs.

## ‚óą Getting Started

### Prerequisites

- **Node.js** `18+`
- **npm** `9+`

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/madhushankarkumar/madhu.dev.git

# 2. Navigate into the project
cd madhu.dev

# 3. Install dependencies
npm install
```

### Development

```bash
# Start local dev server (hot reload enabled)
npm run dev
```

> Opens at `http://localhost:5173`

### Build for Production

```bash
# TypeScript compile + Vite bundle
npm run build
```

> Outputs to `/dist` ‚ÄĒ ready for Vercel, Netlify, or any static host.

### Preview Production Build Locally

```bash
npm run preview
```

## ‚óą Deployment

This project is deployed on **Vercel** with a custom `madhu.me` domain.

**Auto-deploy setup:**
1. Push to `main` branch
2. Vercel detects changes automatically
3. Runs `npm run build`
4. Deploys `/dist` to global edge CDN

**Vercel config (`vercel.json`):**
```json
{
  "buildCommand": "npm run build",
  "outputDirectory": "dist",
  "framework": null
}
```
